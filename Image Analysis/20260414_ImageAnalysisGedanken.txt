Worklflow:
1. RF -> Image : Schallwellen -> Echo -> Signal -> Bild 
2. Segmentation Bild -> Objekte
3. Tracking
4. Optical Flow Pixel -> Bewegung

Pipeline: 
RF -> Reconstruction -> B-mode -> Denoising -> Segmentation (optional) -> optical flow -> Tracking -> Biologische Interpretation

1. klassische Methoden: Thresholding, Kanten-Detektion (Canny), Region-	Growing
